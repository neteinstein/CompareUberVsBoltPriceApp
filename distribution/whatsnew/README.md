# Release Notes

This directory contains release notes for Play Store deployments.

## Structure

Each file should be named with the locale code and `.txt` extension:
- `en-US.txt` - English (United States)
- `es-ES.txt` - Spanish (Spain)
- `fr-FR.txt` - French (France)
- etc.

## Guidelines

- Maximum 500 characters per file
- Focus on user-facing changes
- Be concise and clear
- Update before each Play Store release

## Example

```
Bug fixes and performance improvements.
New feature: Dark mode support.
```
